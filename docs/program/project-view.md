:::info
프로젝트 뷰는 데이터를 체계적으로 관리하는 탐색 기능을 제공합니다.
:::

## 1. 프로젝트 뷰의 구성

프로젝트 뷰는 필드와 슬라이드로 구성되어 있습니다.
* **필드 (field)** : 반입한 데이터의 필드를 의미
* **슬라이드 (Slide)** : 사용자가 분석하면서 생성된 창을 의미합니다.

![프로젝트 뷰](/img/screenshots/project_view.png) 
![프로젝트 슬라이드](/img/screenshots/project_slide.png)


* **프로젝트 트리 :** 생성된 프로젝트와 그 하위의 필드 목록이 트리 구조로 표시됩니다.
* **컨텍스트 메뉴 :** 프로젝트나 필드 항목을 마우스 오른쪽 버튼으로 클릭하면 프로젝트 이름 변경, 닫기, 요약 정보 보기 등의 추가 메뉴가 열립니다.

---

## 2. 컨텍스트 메뉴

프로젝트 또는 필드 값 위에서 마우스 오른쪽 버튼을 클릭하여 컨텍스트 메뉴를 실행할 수 있습니다.
선택된 프로젝트 또는 필드에 기능이 적용됩니다.

### 2.1. 프로젝트 컨텍스트 메뉴

![프로젝트 뷰](/img/screenshots/project-view-project-contextmenu.png)

| 메뉴항목                | 동작                                          |
| :---------------------- | :-------------------------------------------- |
| **Change Project Name** | 클릭하여 프로젝트 이름을 변경합니다.          |
| **View Summary**        | 선택된 프로젝트의 요약 정보를 볼 수 있습니다. |
| **Close Project**       | 선택된 프로젝트를 닫습니다.                   |

### 2.2. 필드 컨텍스트 메뉴

![프로젝트 뷰](/img/screenshots/project-view-field-contextmenu.png)

| 메뉴항목            | 동작                                               |
| :------------------ | :------------------------------------------------- |
| **View List Chart** | 클릭하여 선택된 필드를 기반으로 차트를 생성합니다. |

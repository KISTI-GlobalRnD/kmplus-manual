툴바에서는 자주 사용하는 기능에 대하여 바로 가기 버튼을 제공합니다.
:::

## 툴바의 구성

![툴바](/img/screenshots/toolbar.png)


| 메뉴 항목                                        | 동작 (Action)                                              |
| :----------------------------------------------- | :--------------------------------------------------------- |
| **Create a project**                             | 버튼을 클릭하여 프로젝트 생성을 실행합니다.                |
| **Save as**                                      | 버튼을 클릭하여 현재 프로젝트를 저장합니다.                |
| **New Dataset**                                  | 버튼을 클릭하여 프로젝트 합집합(Union)/교집합을(Intersection) 실행합니다.       |
| **Natural Language Processing**                  | 버튼을 클릭하여 한글 명사, 영문 명사/구 추출을 실행합니다. |
| **Chart**                                        | 버튼을 클릭하여 차트를 생성합니다.                         |
| **Bind Fields**                                  | 버튼을 클릭하여 선택한 필드들을 한 개의 필드로 구성합니다.     |
| **2D Matrix**                                    | 버튼을 클릭하여 2D 행렬을 생성합니다.                    |
| **Multi List**                                   | 버튼을 클릭하여 멀티 리스트를 생성합니다.                    |

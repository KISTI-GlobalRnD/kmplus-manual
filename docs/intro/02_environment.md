## 1. 시스템 요구사항

* KM+는 개인용 PC에서 작동하는 독립형(Stand‐alone) 소프트웨어입니다.
  
* KM+가 효과적으로 작동되기 위한 PC시스템의 최소 사양은 다음과 같습니다.
  
  * CPU : Pentium 4 이상
  * OS 환경 : Windows
  * 메모리 : 8 GB 이상
  * 설치 사이즈 : 500 MegaByte
    
* KM+는 별도의 설치 과정 없이 다운로드 받은 파일을 압축 해제 후 KnowledgeMatrixPlus.exe 를 실행하여 구동합니다.
  
   1. 제공된 압축 파일을 다운로드합니다.
   2. 원하는 폴더에 압축을 해제합니다.
   3. 폴더 내 `KnowledgeMatrixPlus.exe` 실행 파일을 클릭하여 프로그램을 시작합니다.

## 2. 데이터 처리 가능 범위

* 메모리 4,096MB (4GB)까지 사용 가능
  
* 반입 레코드(Importing Raw Record) 수 제한
  * 안정적인 레코드(Raw Record) 수는 WoS Plain Text 파일 기준 25,000건 내외입니다.
    
* Excel Export 제한
  * Excel 워크시트 제한으로 인하여, 다음을 초과하는 데이터의 Excel Export 시에 에러 발생합니다.
  * .xlsx : 1,048,576 * 16,384 / .xls : 65,536 x 256
    
* 사용자의 PC 환경에 따라 데이터 처리 속도와 성능이 다를 수 있습니다.

## 1. KnowledgeMatrix Plus 소개

<div className="show-title">

:::info[KnowledgeMatrix Plus]
KnowledgeMatrix Plus (이하 KM+)는 논문, 특허 등 구조화된 텍스트 데이터에서 필요한 정보를 쉽게 추출하여, 데이터 전처리와 네트워크 분석을 효율적으로 지원하는 과학계량 분석 소프트웨어입니다. 본 소프트웨어는 전문적인 지식이 없어도 구조화된 텍스트 정보를 쉽게 분석할 수 있도록 한국과학기술정보연구원(KISTI) 글로벌R&D분석센터에서 주관하여 개발하였습니다.
:::

</div>

### 1.1. 배경 및 목적

논문, 특허 등 텍스트 기반 데이터의 분석에서 일반적으로 데이터 전처리와 분석 도구에 맞는 입력 데이터 준비에 가장 많은 시간이 소요됩니다.
KnowledgeMatrix Plus (KM+)는 이러한 과정을 체계적이고 자동화된 방식으로 지원하여, 연구자가 쉽고 효율적으로 연구 동향 및 네트워크 분석을 수행할 수 있도록 돕습니다. 특히, 다양한 오픈 소스 네트워크 분석 도구 반입에 적합한 데이터를 생성하여 분석 연계를 지원합니다. 또한, 기존의 해외 과학계량 분석 소프트웨어는 한글 처리가 어려웠으나, KM+에서는 한글 데이터 처리를 가능하게 해줍니다.

## 2. 주요 기능

### 2.1. 데이터 전처리

* **형태소 분석:** Apache OpenNLP(영문)와 아리랑 형태소 분석기(한글)를 탑재하여, 일반 텍스트에서 명사(구) 추출
* **데이터 정제:** 그룹 기능을 활용한 데이터 축소 및 확장, 불용어 처리, 스테밍(어근 추출), 시소러스(유의어) 처리를 통해 데이터 표준화
* **중복 제거 및 집합 관리:** 프로젝트 간 합집합, 교집합 만들기 기능을 통해 데이터 비교 분석 지원


### 2.2. 행렬 및 다중 리스트 생성

* **2D Matrix:** 행과 열의 필드를 지정하여 Raw Record 기준의 동시 출현 행렬 생성
* **Multi List:** 3개의 필드를 조합한 리스트를 생성하여 입체적인 데이터 분석 지원

### 2.3. 다양한 포맷으로 데이터 반출

:::note
KM+로 생성된 데이터는 별도의 변환 과정 없이 대표적인 네트워크 분석 및 시각화 소프트웨어에서 즉시 활용 가능합니다.
:::

* **반출 파일 타입:** Excel, CSV, .NET/CLU/VEC (Pajek), .GEXF (Gephi), VOSviewer 등
* **속성 데이터 포함:** 네트워크 분석 시 필요한 노드의 속성(레코드 수) 및 집단(Partition) 정보까지 함께 출력



### 2.4. 데이터 처리 성능

* Web of Science 기준 약 **25,000건**의 논문 데이터를 안정적으로 처리할 수 있으며, 데이터 형식에 따라 최대 수십만 건의 데이터 처리가 가능합니다.
* 단, 개별 PC사양에 따라 데이터 처리 성능은 다를 수 있습니다.

## 3. 주요 활용 프로세스

:::note
KM+를 활용한 데이터 분석은 다음과 같은 흐름으로 진행됩니다.
:::

1. **데이터 반입 (Import):** Import Filter를 활용하여 사전에 정의된 형태의 파일을 반입하거나, 필드화된 엑셀 형식의 파일을 반입합니다.
2. **데이터 전처리 (Pre-processing):** 자연어 처리, 스테밍, 시소러스 적용 등을 통해 분석에 적합한 형태로 데이터를 가공합니다.
3. **리스트 생성 (List):** 선택한 필드별 레코드 수를 리스트로 확인합니다.
4. **행렬 생성 (Matrix Generation):** 분석 목적에 알맞게 행렬이나 멀티리스트를 생성합니다.
5. **데이터 내보내기 (Export):** 생성된 행렬을 Pajek, VOSviewer, Gephi에 반입할 수 있는 파일로 내보내어 네트워크 구조를 시각화하고 심층 분석을 수행합니다.

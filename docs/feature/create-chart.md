## 1. 차트 생성 방법 (메뉴 바)

**[STEP 1]** 차트 종류 및 옵션 선택

1. **Graph Type**을 선택합니다.
2. **Sorting Method**를 선택합니다.
3. **Upper Range Value** 값을 지정합니다. (Optional)
4. **View 3D** 여부를 선택합니다. (Optional)
5. **View Examples**(Legend Box) 노출 여부를 선택합니다. (Optional)
6. **Next** 버튼을 클릭합니다.

![차트 선택 화면](/img/screenshots/create-chart-start-1.png)

**[STEP 2]** 차트를 생성할 필드 선택

1. 차트를 그릴 필드를 선택합니다.
2. **Finish** 버튼을 클릭합니다.

![필드 선택 화면](/img/screenshots/create-chart-select-field.png)

## 2. 차트 생성 방법 (필드 리스트 툴바)

**[STEP 1]** 차트 종류 및 옵션 선택

1. **Graph Type**을 선택합니다.
2. **Sorting Method**를 선택합니다.
3. **Upper Range Value** 값을 지정합니다. (Optional)
4. **View 3D** 여부를 선택합니다. (Optional)
5. **View Examples**(Legend Box) 노출 여부를 선택합니다. (Optional)
6. **Finish** 버튼을 클릭합니다.

![차트 선택 화면](/img/screenshots/create-chart-start-2.png)

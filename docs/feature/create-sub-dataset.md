:::info
* 생성된 리스트에서 그룹이나 아이템을 선택하여 데이터 서브셋을 생성하는 기능입니다.
* 데이터에서 일부의 데이터만 선택하여 따로 분석할 때 활용할 수 있습니다.
    * (활용 예) 특정 기간, 국가, 기관, 분야의 데이터 분석
* [`메인 작업 영역 > 필드 리스트 뷰 : 툴바`](../program/workspace/fieldlistview.md#toolbar) 를 통해 실행할 수 있습니다.

![필드 리스트-툴바](/img/screenshots/fieldlistview-toolbar.png)
:::

## 1. 서브 데이터셋 생성 방법

**[STEP 1]** 리스트에서 행을 선택합니다.

**[STEP 2]** '상단 필드 리스트 뷰 : 툴바'나 오른쪽 마우스를 클릭할 때 생성되는 컨텍스트 뷰에서 `Create Sub-dataset`을 선택합니다.
* with records Selected : 선택된 레코드를 기준으로 서브셋을 생성합니다.
* with Group Selected : 선택된 그룹을 기준으로 서브셋을 생성합니다.

![서브셋 생성](/img/screenshots/create-sub-dataset-1.png)

**[STEP 3]** 새로운 프로젝트명을 입력합니다.

![서브셋 생성](/img/screenshots/create-sub-dataset-2.png)

## 2. 서브 데이터셋 생성 결과

* 프로젝트 뷰에 새로운 프로젝트가 생성됩니다.
* 메인 작업 영역에 Summary Sheet가 생성됩니다.


![서브셋 생성 결과](/img/screenshots/create-sub-dataset-result.png)

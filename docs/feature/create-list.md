:::info
* 리스트는 필드별 아이템 수와 레코드를 보여줍니다.
* KM+ 활용의 가장 기본적인 기능입니다.
:::

## 리스트 생성 방법

* 프로젝트 뷰에서 각 필드를 더블 클릭하여 리스트를 생성합니다.
* (리스트 예시) 프로젝트 뷰에서 저자 소속 국가 (Author Affiliation (Countries/Regions))를 선택하여 리스트를 생성합니다.
* 각 열 헤더를 더블클릭하여 정렬합니다.

![리스트 생성](/img/screenshots/create-list.png)

* **Records** : 해당 값이 포함된 레코드 수를 의미합니다.
* **Instances** : 해당 값의 갯수를 의미합니다.

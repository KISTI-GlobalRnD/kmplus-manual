:::info
* 여러 개의 필드 데이터를 병합하여 새로운 필드를 생성합니다.
* [`메뉴바 : Pre-processing > Bind Fields`](../program/menu-bar.md#pre-processing)를 통해 기능을 실행할 수 있습니다.

![pre-processing](/img/screenshots/menubar-preprocessing.png)

:::


## 필드 병합 방법

**[STEP 1]** 팝업창에서 병합할 필드를 선택합니다.

**[STEP 2]** **Field Name**을 입력합니다.

**[STEP 3]** **Finish**를 클릭합니다.

![바인드 필드 선택](/img/screenshots/bind-fields.png)


## 필드 병합 결과

* 선택한 프로젝트 필드 목록 맨 끝에 병합된 필드가 추가됩니다.

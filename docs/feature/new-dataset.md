:::info
* 프로젝트를 선택하여 합집합 또는 교집합을 수행하여 새로운 프로젝트를 생성합니다.
* 이 기능은 KM+에 2개 이상의 프로젝트가 생성되어 있어야 합니다.
* 프로젝트 합집합 또는 교집합은 [`메뉴 바 : Pre-processing > New dataset`](../program/menu-bar.md#pre-processing)에서 실행하실 수 있습니다.

    ![pre-processing-new dataset](/img/screenshots/menubar-newdataset.png)
:::

---

## 1. 프로젝트 합집합 (Project Union)

1. 왼쪽 프로젝트 목록에서 합집합을 적용할 프로젝트를 선택합니다.
2. 중앙의 ▶ 버튼을 눌러 합집합 대상 프로젝트 목록에 추가합니다.
3. **Project Name** 을 입력합니다.
4. **Finish** 버튼을 클릭합니다.

![프로젝트 합집합](/img/screenshots/project-union.png)

---

## 2. 프로젝트 교집합 (Project Intersection)

1. 왼쪽 프로젝트 목록에서 교집합을 적용할 프로젝트를 선택합니다.
2. 중앙의 ▶ 버튼을 눌러 교집합 대상 프로젝트 목록에 추가합니다.
3. 오른쪽 프로젝트 목록에서 교집합 대상이 될 **필드를 선택**합니다.
4. **Project Name** 을 입력합니다.
5. **Finish** 버튼을 클릭합니다.

![프로젝트 교집합](/img/screenshots/project-intersection.png)

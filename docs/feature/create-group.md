:::info
* 생성된 리스트에서 데이터 일부를 그룹으로 지정하여 새로운 데이터셋으로 생성할 수 있습니다.
* [`메인 작업 영역 > 필드 리스트 뷰 : 툴바`](../program/workspace/fieldlistview.md#toolbar) 를 통해 실행할 수 있습니다.

![필드 리스트-툴바](/img/screenshots/fieldlistview-toolbar.png)
:::

## 1. 그룹 생성

1. 리스트에서 행을 선택합니다.
2. `Add Selection to New Group`을 선택합니다.
   * 필드리스트 뷰 상단의 `Add Selection to New Group`버튼을 클릭하거나
   * 선택한 영역에서 오른쪽 마우스를 클릭하여 나오는 컨텍스트 메뉴의 `Add Selection to New Group`을 선택합니다.
3. Create New Group 창에 그룹명을 입력하고 `OK`버튼을 클릭합니다.

![그룹 생성](/img/screenshots/create-group.png)

4. 그룹 생성 결과를 확인합니다.
   * 리스트에 입력한 그룹명 필드가 생성되고 선택한 아이템이 체크박스에 표시됩니다.
   * 프로젝트 뷰에서 해당 필드의 하위 구조로 그룹명이 생성된 것을 확인합니다.

![그룹 생성 결과](/img/screenshots/create-group-result.png)

## 2. 그룹 아이템 추가/제거

1. 체크박스를 클릭하여 그룹에 아이템을 추가하거나 제거할 수 있습니다.
2. 필드리스트 뷰 상단의 `Add to group`버튼을 활용할 수 있습니다.

## 3. 그룹 관리

1. 필드리스트 뷰 상단의 `Manage groups`버튼으로 그룹 관리를 실행합니다.

![그룹 관리](/img/screenshots/create-group-manage.png)

* **Add**: Group Name에 그룹명을 입력하여 새로운 그룹을 생성합니다.
* **Delete**: Group List에서 선택한 그룹을 삭제합니다.
* **Rename**: Group List에서 그룹을 선택하고 그룹명을 변경합니다.

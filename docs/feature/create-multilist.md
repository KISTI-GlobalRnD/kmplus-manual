:::info
* 3개의 필드를 선택하여 다중 리스트를 생성하여 그 결과를 표로 출력하거나, 엑셀 파일로 다운로드 합니다.
* [`메뉴바 : Pre-processing > Multi List`](../program/menu-bar.md#analysis) 또는 [`메인툴바`](../program/toolbar.md)를 통해 실행할 수 있습니다.

   ![analysis](/img/screenshots/menubar-analysis.png)
:::


## 1. 멀티 리스트 대상 필드 선택 

* 대상이 되는 필드를 선택합니다.
* 멀티 리스트 생성 예시 : WoS (Plain Text) 데이터 `[저자 소속 국가 x 출판연도 x 분야]`

![멀티 리스트 생성 화면](/img/screenshots/multilist-select.png)

## 2. 멀티 리스트 생성 결과 확인

### 2-1. 미리보기 (Preview)

* **Preview** 버튼을 클릭하여 멀티 리스트 생성 결과를 KM+ 메인 작업 영역에서 확인합니다.

![멀티 리스트 생성 preview 화면](/img/screenshots/multilist-result-preview.png)

### 2-2. 다운로드 (Download)

* **Download** 버튼을 클릭하여 멀티 리스트 생성 결과를 엑셀 형식으로 PC에 다운로드합니다.
* 조합의 수가 많아 화면에서 결과 확인이 어려울 때 활용합니다.

![멀티 리스트 생성 download 화면](/img/screenshots/multilist-result-download.png)

![멀티 리스트 생성 download 결과](/img/screenshots/multilist-result-download-excel.png)
